Moottorin kunnonseurantasovellus

Tämä Python-sovellus simuloi moottorin toimintaa ja seuraa sen kuntoa reaaliajassa.  
Ohjelma näyttää tärkeimmät mittausarvot (lämpötila, tärinä, kierrosnopeus, öljynpaine ja jännite) sekä hälyttää poikkeamista.

Sovellus toimii prototyyppinä, ja siihen on toteutettu virtuaalianturit, jotka tuottavat simuloitua anturidataa.

2. Käyttötarkoitus

- Moottorin kunnon seuraaminen ja valvonta
- Huoltotarpeen ennakointi


3. Päätoiminnot

1. Reaaliaikainen mittausten seuranta (lämpötila, tärinä, kierrosnopeus, öljynpaine, jännite)
2. Hälytysjärjestelmä poikkeamien havaitsemiseksi
3. Graafinen visualisointi mittausten kehityksestä ajan kuluessa
4. Lokitiedoston tallennus ja tarkastelu
5. Mahdollisuus kuittaa hälytykset ja käynnistää/pysäyttää simulaatio
